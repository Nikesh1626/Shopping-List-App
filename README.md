# Shopping List App

**Engineered a native Android application** using Kotlin and Jetpack Compose to streamline personal shopping management through a highly responsive, single-screen architecture.

## 🚀 Features

- **Live State Management**: Robust state management system to drive real-time data synchronization across the UI.
- **Dynamic Search & Filtering**: Instant search filtering and dynamic category sorting.
- **Item Management (CRUD)**: Intuitive, custom-built dialogs to efficiently input item details, assign categories, and track item quantities.
- **Live Inventory Counting**: Real-time updates on the total number of items in your list.
- **Modern UI/UX**: Engaging and cohesive user experience with modular, reusable UI components including interactive category grids and custom navigation elements adhering to Material Design principles.

## 🛠️ Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM (Model-View-ViewModel)
- **State Management**: Compose State & ViewModels
